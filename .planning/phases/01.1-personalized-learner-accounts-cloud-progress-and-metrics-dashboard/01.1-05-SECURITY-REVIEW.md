## Findings

### SEC-01 â€” `/api/challenges/assess` lacks the exact-origin denial applied to `/api/progress` although it is now a cloud-write surface (Severity: Low)

- **Where:** `src/app/api/challenges/assess/route.ts` (no `assertSameOrigin`) vs `src/app/api/progress/route.ts` (`assertSameOrigin` on GET/POST/DELETE).
- **Analysis:** Since Plan 01.1-03, an Authorization-bearing assess request triggers an authenticated Firestore write through `syncAssessment`. The phase's stated containment is that cross-origin browser requests "must never reach the data boundary" â€” the progress route enforces it, the assess-with-token path does not. This is **not** exploitable as classical CSRF (the Firebase ID token is not an ambient credential; a cross-site attacker cannot attach the victim's token), which caps severity at Low. It is a defense-in-depth parity gap on the newest write path, and the cheapest place to stop a future regression (e.g., if any ambient credential is ever introduced).
- **Reproduction (unit-level):** Invoke the assess POST handler with `Origin: https://evil.example`, matching valid body, and a valid emulator-verified token: the cloud write proceeds. The identical request against `/api/progress` returns 403 `CROSS_ORIGIN`.
- **Expected disposition:** Apply the same exact-origin check to the assess route (Origin-absent requests continue to pass, so non-browser clients, smokes, and route tests are unaffected), with an adversarial regression in `src/app/api/challenges/routes.test.ts`.

### SEC-OBS-A â€” Tolerant cloud-document parsing degrades unparseable documents to the empty profile (Informational, no action requested)

`parseCloudDocument` in `progress-repository.ts` returns the empty profile for unrecognizable stored data, and the next write persists the re-derived state. Because Firestore is writable only via the server (deny-all rules, Admin-only), an unparseable document implies operator-level tampering or schema migration, and fail-open-to-empty is the correct availability trade for learning data (never fabricates progress, never blocks the learner). Recorded so the disposition is a conscious decision, not an oversight.
